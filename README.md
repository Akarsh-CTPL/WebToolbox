# 🛠️ Web Toolbox | CTPL 

A collection of high-efficiency, self-service automation tools developed for the **CTPL Team**. These utilities are designed to replace manual processes with instant, rock-solid digital solutions.

> **Status:** Production Bridge. These tools are active and functional while our primary integrated application remains in development.

---

## 🚀 The Tools

### 📄 [Certificate Generator](./installation-certificate.html)
Generate professional PDF installation and fitment certificates. 
* **Key Features:** RTO compliance, instant preview, and Google Sheets integration for data logging.

### 📅 [Plan Calculator](./planCalculator.html)
Smart eligibility checker for RC validity.
* **Key Features:** Precise age calculation (Years/Months/Days) and automatic 1-year vs 2-year plan detection based on the 8-year rule.

### 🏷️ [SLD Labels](./custom-labels.html)
Bulk label generator for hardware units.
* **Key Features:** Zone-based generation for ETL/MTL series, dynamic QR codes, and IMEI sticker formatting.

---

## 🛠️ Tech Stack
* **Frontend:** HTML5, Bootstrap 5, Inter Font
* **Logic:** Vanilla JavaScript (ES6+)
* **Libraries:** jsPDF (PDF Generation), QRCode.js (Dynamic QR)
* **Backend:** Google Apps Script (Data Sync)

These tools were built so you don't have to wait for manual processing.
